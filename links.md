## If you're interested in Stats, Data Science, and Programming

### Learning Statistics and Data Science

#### [An Introduction to Statistical Learning](https://www-bcf.usc.edu/~gareth/ISL/) <br>
Free book that provides an introduction to many modeling and prediction methods with examples in R. Undergraduate level, only requires a prior class in linear models.

#### [Kaggle](https://www.kaggle.com/kernels) <br>
Data science hub where you can find competitions to compete in, datasets to practice on, and notebooks in R and Python to learn from.


## Learning R

#### [cRan Introduction to R](https://cran.r-project.org/doc/manuals/r-release/R-intro.pdf) <br>
Extensive introduction to R.

#### [R Style Guide](http://adv-r.had.co.nz/Style.html) <br>
Guide to proper conventions for coding in R by Hadley Wickham, the chief scientist of RStudio.

#### [My R Guides](https://stevebroll.github.io/guides/)
Individual pdf guides covering R functions frequently used in introductory statistics courses:  


### Learning Git and GitHub
#### [Git with RStudio](http://happygitwithr.com/big-picture.html) <br>
Introduction to Git with a beginner's guide to connecting to GitHub via RStudio projects.

#### [Git from Command Line](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git) <br>
Guide to getting started with Github using command-line tools in Windows, Mac, and Linux.

## If you're a Texas A&M student

### Degree Planning
#### [Aggie Scheduler](http://www.aggiescheduler.com/) <br>
Schedule planner with a course search, information for each class section, and clean graphical view, created by alum Jake Leland.

#### [General Undergraduate Catalog](http://catalog.tamu.edu/undergraduate/course-descriptions) <br>
Directory of undergraduate course lists for every prefix.

#### [B.S. Statistics Flowchart](https://www.stat.tamu.edu/wp-content/uploads/2016/03/bs_flowcharts.pdf) <br>
Overview of statistics major with a flowchart showing the necessary order of math/stat classes to meet prereqs and graduate on time.

#### [Minor in Statistics](https://www.stat.tamu.edu/academics/minor/) <br>
Requirements for minor in statistics.

#### [STAT Undergraduate Courses](http://catalog.tamu.edu/undergraduate/science/statistics/#coursestext) and [STAT Graduate Courses](http://catalog.tamu.edu/graduate/colleges-schools-interdisciplinary/science/statistics/#coursestext) <br>
STAT-prefix undergraduate and graduate course lists, respectively.

#### [Texas Common Course Numbering System](http://catalog.tamu.edu/undergraduate/appendices/texas-common-course-numbering-system/) and [Transfer Course Equivalency](https://compass-ssb.tamu.edu/pls/PROD/bwxkwtes.P_TransEquivMain?te_type=crse) <br>
Find out whether TAMU has equivalents to classes from other institutions.
