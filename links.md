## If you're interested in Stats, Data Science, and Programming

### Learning Statistics and Data Science

#### [An Introduction to Statistical Learning](https://www-bcf.usc.edu/~gareth/ISL/) <br>
Free book that provides an introduction to many modeling and prediction methods with examples in R. Undergraduate level, only requires a prior class in linear models.

#### [Kaggle](https://www.kaggle.com/kernels) <br>
Data science hub where you can find competitions to compete in, datasets to practice on, and notebooks in R and Python to learn from.


## Learning R

#### [cRan Introduction to R](https://cran.r-project.org/doc/manuals/r-release/R-intro.pdf) <br>
Extensive introduction to R.

#### [R Style Guide](http://adv-r.had.co.nz/Style.html) <br>
Guide to proper conventions for coding in R by Hadley Wickham, the chief scientist of RStudio.
